Sobre o Projeto

Este projeto tem como objetivo facilitar a geração de relatórios de pedidos com base em filtros de datas e status. Ele foi desenvolvido para fins de aprendizado e integração de diferentes tecnologias.

Funcionalidades:

  • Filtro por data de início e fim
  • Filtro por status do pedido
  • Exportação dos dados para Excel

Tecnologias Utilizadas

  • Python
  • Flask (framework web)
  • PostgreSQL (banco de dados relacional)
  • SQLAlchemy (ORM)
  • Pandas (manipulação de dados)
  • openpyxl (exportação para Excel)
  • HTML5 + CSS3
  • Jinja2 (template engine do Flask)
  • dotenv (variáveis de ambiente)

Este sistema se conecta a um banco de dados PostgreSQL hospedado na nuvem para consultar os dados em tempo real.
